# Filmorate — шаблон командного проекта

Этот репозиторий — **шаблон без кода**, только структура и файлы-заглушки.
Используйте его для командной работы: ветки `main` и `develop`, — а также CI и шаблоны задач/PR.

## Старт
1. Создайте ветку `develop` от `main`.
2. Включите правила защиты веток (2 approvals, запрет прямых пушей).
3. Скопируйте проект-доску GitHub Projects и назначьте Story points и Исполнителей.
4. Начните декомпозицию и распределение задач.

## Модули (папки)
- `controller` — REST-контроллеры.
- `service` — бизнес-логика.
- `repository` — доступ к БД.
- `model` — доменные сущности.
- `mapper` — конвертация DTO ↔ модель.
- `exception` — обработка ошибок.
- `config` — конфигурация Spring/приложения.
- `util` — утилиты.
- `resources` — конфигурация (например, `application.yml`).

> Вся реализация появится позже: сейчас в Java-файлах есть только один русский комментарий.
