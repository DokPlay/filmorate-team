# Проект-доска (GitHub Projects) — как скопировать

1. Откройте исходную доску и нажмите **Make a copy**.
2. Отметьте опцию **Draft issues will be copied if selected**.
3. Выберите свой аккаунт и завершите копирование.
4. Сделайте проект **Public** и пригласите команду в **Manage access**.
5. Создайте числовое поле **Story points** и заполните, как на исходной доске.
6. Распределите задачи, у каждой должен быть **Assignee** и **Status**.
